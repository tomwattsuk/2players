# 2players

2players is a real-time multiplayer gaming platform that currently features Tic-Tac-Toe and Connect Four. This application allows users to register, log in, select games, and play against other online players in real-time.

## Features

- User authentication (register, login, logout)
- Game selection
- Real-time gameplay for Tic-Tac-Toe and Connect Four
- Matchmaking system

## Technologies Used

- Frontend: React.js
- Backend: Node.js with Express.js
- Database: MongoDB
- Real-time Communication: Socket.IO

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)
- MongoDB (v4.0.0 or later)
